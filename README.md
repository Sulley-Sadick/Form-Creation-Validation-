# Project Title

Form Validation

This project is about learning how to validate input field when working with form elements.

## Live Demo

http://127.0.0.1:8080/

## Usage

1. Fill input fields
2. Press enter or Register button
3. if user's information is invalid, show error message
4. if user's information is valid, show success message

1. HTML
2. CSS
3. Javascript
